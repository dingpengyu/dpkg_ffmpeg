Automated build are available at [Releases](https://github.com/link-u/dpkg_ffmpeg/releases).

## How to build manually

```bash
git clone --recurse-submodules --recursive git@github.com:link-u/dpkg_ffmpeg.git
cd dpkg_ffmpeg
bash scripts/all.sh
```
